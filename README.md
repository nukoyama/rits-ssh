# rits-ssh
ritsのリモートサーバへのssh接続


## mac

ritsにvpn接続する。
- 参照： https://secure.ritsumei.ac.jp/rainbow/service-vpn/

ターミナルを起動して
```sh
ssh <Rainbow ID>@remote.ritsumei.ac.jp
```
と打つ。

何か聞かれるので `yes` と入力。
その後Rainbowのpasswordを入力すればritsのlinuxに入れる。
